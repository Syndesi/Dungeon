Class: ProgressBar

ProgressBar

Creates a ProgressBar which can be used like any PIXI.Container.

new ProgressBar(width, min, max, progress, color)

primitives/progressBar.js, line 10

Creates a new ProgressBar

Name Type Default Description
width int 80

the width of this component
min int 0

the smallest value this bar can show
max int 100

the highest value this bar can show
progress int 33

the current value
color string Blue

one of these: Red, Orange, Yellow, Green, Blue, Purple, Black, White
Example
var bar = new ProgressBar(100, 0, 100, 0);
// the blue bar will now be 90% 
bar.setProgress(90);
